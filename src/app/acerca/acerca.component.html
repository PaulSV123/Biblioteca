Acerca del proyecto Biblioteca de programación

Sobre el proyecto visto en esta página se podría decir que será un lugar donde se guardarán los diferentes tipos de libros de interés de un lector común hasta uno experimentado sin la necesidad de evitar o bloquear los accesos a estos libros. El proyecto será realizado en Angular.js para la parte de front-end, Node.js para la parte de backend y la base de datos será manejado en Mysql.
